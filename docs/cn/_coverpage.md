# sureness <small>0.4.4 for jvm</small>  

> 面向`restful api`的高性能认证鉴权框架.  

- 基于 `rbac` (用户-角色-资源)主要关注于对 `restful api` 的安全保护    
- 无特定框架依赖(`springboot,quarkus,javalin,ktor`等集成样例)  
- 支持主流`http`容器 `servlet` 和 `jax-rs` 
- 支持动态权限配置   
- 支持多种认证策略
