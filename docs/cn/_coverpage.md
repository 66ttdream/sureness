# sureness <small>1.0.3 for jvm</small>  

> 面向`REST API`的高性能认证鉴权框架   

- 基于`RBAC`, 多种认证方式(`JWT,Basic,Digest Auth`)  
- 无`Web`框架依赖，极快的性能(不吹牛有基准测试哦)    
- 支持`Websocket` ,`Http Servlet`和`JAX-RS`  
- 支持动态权限, 易用急速扩展
